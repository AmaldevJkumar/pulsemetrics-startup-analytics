## Overview
PulseMetrics is an end-to-end analytics platform designed to simulate how a real SaaS startup tracks growth, product usage, and revenue. The project demonstrates analytics engineering, product analytics, and business intelligence best practices using SQL, dbt, Python, and Power BI.

This project is being built incrementally and follows a production-style analytics architecture.

---

## Business Objectives
- Track user growth, activation, retention, and churn
- Monitor revenue metrics such as MRR, ARR, and LTV
- Identify funnel drop-offs and user behavior patterns
- Enable leadership to make data-driven product decisions
